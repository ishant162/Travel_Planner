# AI Travel Planner - Multi-Agent Itinerary Generator

An intelligent multi-agent travel planning system that leverages **AutoGen** and **OpenRouter API** to generate comprehensive, personalized travel itineraries. This repository orchestrates specialized AI agents—including a Travel Planner and Researcher—that collaboratively discover destinations, research attractions, and craft detailed day-by-day travel plans.

## Key Features

- Multi-agent architecture with planner and researcher roles
- Powered by OpenRouter API for flexible LLM access across multiple models
- Built with Microsoft AutoGen for robust agent orchestration and communication
- Generates comprehensive travel itineraries with local insights and activities
- Includes destination research, accommodation suggestions, and activity recommendations

## Architecture

- **Travel Planner Agent**: Creates structured travel plans based on user preferences, dates, and budget
- **Researcher Agent**: Discovers authentic local experiences, attractions, and cultural insights
- Agent collaboration via AutoGen's conversation patterns for refined itineraries

## Tech Stack

- **Framework**: Microsoft AutoGen
- **LLM Provider**: OpenRouter API (supports multiple model providers)
- **Language**: Python-based with modular agent design

## Use Cases

Perfect for travel enthusiasts, travel agencies, or anyone building automated trip planning tools that require intelligent, context-aware itinerary generation with minimal manual input.
